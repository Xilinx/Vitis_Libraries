L\-Z77 compression kernel takes the raw data as input and compresses the data in block based fashion and writes the output to global memory. L\-Z77 is a byte based compression scheme. The resulting output from this kernel is represented in packet form of 32bit length $<$Literal, Match Length, Distance$>$. It also generates output of literal and distance frequencies for dynamic huffman tree generation. The output generated by this kernel is referred by Tree\-Gen and Huffman Kernels.
