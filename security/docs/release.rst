.. toctree::

1.0
----

The 1.0 release provides a range of HLS primitives for:

 * Symmetric Block Cipher
 * Symmetric Stream Cipher
 * Cipher Operation Mode
 * Message Authentication code(MAC)
 * Hash Function
 * Asymmetric Cryptography
