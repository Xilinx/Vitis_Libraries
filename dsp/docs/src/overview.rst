============
Requirements
============

Software Platform
=================

This library is designed to work with AMD Vitis |trade|, and therefore inherits the system requirements of Vitis and XRT.

Supported operating systems are RHEL/CentOS 7.9, RHEL 8.8, RHEL 8.9, RHEL 8.10, RHEL 9.2, RHEL 9.3, and RHEL 9.4, Ubuntu 20.04.5 LTS, 20.04.6 LTS, 22.04.1 LTS, 22.04.2 LTS, 22.04.3 LTS, and 22.04.4 LTS, as well as AlmaLinux OS ver. 8.7, 8.10, 9.1, and 9.4.


PCIe Accelerator Card
=====================

Hardware modules and kernels are designed to work with AMD Alveo |trade| U200 and U280 cards.
