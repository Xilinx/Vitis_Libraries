# Impulse Test
Runs an impulse test on FFT, to launch it cahnge to folder "impulse_test" , set_env and launch "run_test.tcl" using vitis_hls

1. cd impulse_test
2. source set_env.sh
3. vitis_hls -f run_hls.tcl
